# 💱 Currency Converter
A real-time currency converter web app built with HTML, CSS, and JavaScript. Converts currencies using live exchange rate API with country flags.

🚀 **Live Demo:** https://currency-converter-harshita22.vercel.app

## ✨ Features
- Real-time currency conversion using Exchange Rate API
- Country flags displayed for each currency
- Supports multiple world currencies
- Clean and minimal UI
- Instant conversion on input

## 🛠️ Built With
- HTML5
- CSS3
- JavaScript (Vanilla)
- Exchange Rate API

## 📁 Project Structure
currency-converter/
├── index.html     # Main HTML file
├── style.css      # Styles
├── app.js         # API logic and conversion
└── codes.js       # Currency codes and country flags

## 🚀 How to Run Locally
git clone https://github.com/harshita047/currency-converter.git
cd currency-converter
Open `index.html` in any browser.
